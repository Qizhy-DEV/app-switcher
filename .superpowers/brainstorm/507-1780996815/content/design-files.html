Design — Cấu trúc file & Deployment

Tổ chức source code và quy trình từ push đến production

Cấu trúc thư mục

app-switcher/
├── src/
│ ├── app-switcher.ts ← custom element chính (<app-switcher>)
│ ├── app-trigger.ts ← icon 9 chấm
│ ├── app-popover.ts ← grid 4 cột
│ ├── app-service.ts ← fetch + cache apps.json
│ └── index.ts ← entry point, export
├── public/
│ └── apps.json ← danh sách app (edit ở đây)
├── .gitlab-ci.yml ← CI/CD pipeline
├── vite.config.ts
└── package.json

Vite build output

dist/app-switcher.js ~15KB gzip · component
dist/apps.json config · copy từ public/

Vite build với lib mode — output là 1 file JS duy nhất, không có HTML wrapper.
